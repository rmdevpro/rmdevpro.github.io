Privacy Policy

Last updated: March 26, 2026

Overview

This privacy policy describes how J. Morrissette ("I", "me") handles information in connection with the Joshua26 system alerting service. This service sends automated SMS notifications about system events to the system administrator (me). It is not a consumer-facing service.

Information Collected

The alerting service processes the following information:

  • Phone numbers: The administrator's phone number, used solely to deliver system alert messages.
  • Alert content: Automated system status messages generated by the Joshua26 infrastructure (e.g., health status, pipeline errors, scheduled reports). These contain no personal information about end users.

How Information Is Used

Phone numbers and alert content are used exclusively to deliver system notifications to the administrator. No information is:

  • Shared with third parties (except Twilio, the SMS delivery provider)
  • Used for marketing purposes
  • Sold or transferred
  • Used to build user profiles

Data Retention

Alert history (event type, message content, delivery status) is stored in the system's internal database for operational diagnostics. Phone numbers are stored only in the system configuration file. No data is retained by third parties beyond Twilio's standard message delivery logs.

Third-Party Services

SMS messages are delivered via Twilio. Twilio's privacy policy governs their handling of message delivery data.
