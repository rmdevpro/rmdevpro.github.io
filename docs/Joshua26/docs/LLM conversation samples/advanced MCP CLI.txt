Here is the architecture for Persistent Sharded Memory:

The "Vault & Workbench" Architecture
You have two locations for every session:

The Vault (Persistent): A folder on your real hard drive (e.g., /storage/memory/agent_884/). This holds the "Frozen" state.

The Workbench (Ephemeral): The folder in RAM (e.g., /dev/shm/agent_884_run_1/). This is where the CLI actually runs.

The Workflow: Hydrate -> Execute -> Dehydrate
Every time an agent wants to "speak" or "do" something, your wrapper script performs these 4 steps:

1. Hydrate (Checkout)
The wrapper checks if agent_884 exists in The Vault.

If Yes: It copies the entire configuration folder (state.json, history.db, etc.) from Disk to RAM.

If No: It copies a "Template" (containing just the API Key/Auth) to RAM to start fresh.

2. Execute (The Action)
You run the CLI command pointing CLAUDE_CONFIG_DIR to the RAM location.

The CLI reads the history (that you just copied there).

The CLI thinks, generates a response, and updates the state.json in RAM.

Crucial: No file locks occur because this RAM folder is unique to this specific process.

3. Dehydrate (Check-in)
Once the CLI process exits successfully (return code 0), your wrapper copies the updated state files from RAM back to The Vault on the disk.

Optimization: You typically only need to copy back the state.json or SQLite DB file, but copying the whole folder is safer.

4. Cleanup
Delete the RAM folder.

The Implementation (Python Wrapper)
Here is the robust Python logic to handle this "Teleporting State":

Python
import os
import shutil
import subprocess
import uuid

# Configuration
VAULT_ROOT = "/storage/sessions"       # Permanent HDD storage
TEMPLATE_DIR = "/storage/auth_template" # Contains just claude.json (Auth)
RAM_ROOT = "/dev/shm/claude_execution"  # Fast RAM disk

def run_persistent_agent(session_id, user_prompt):
    # 1. Define Paths
    vault_path = os.path.join(VAULT_ROOT, session_id)
    execution_path = os.path.join(RAM_ROOT, f"{session_id}_{uuid.uuid4()}")

    try:
        # --- PHASE 1: HYDRATION (Checkout) ---
        if os.path.exists(vault_path):
            # Resume: Copy existing history to RAM
            shutil.copytree(vault_path, execution_path)
        else:
            # New Session: Copy auth template to RAM
            shutil.copytree(TEMPLATE_DIR, execution_path)

        # --- PHASE 2: EXECUTION (Run) ---
        # Run Claude pointing to the RAM location
        env = os.environ.copy()
        env["CLAUDE_CONFIG_DIR"] = execution_path
        
        # We use -p to run headless
        result = subprocess.run(
            ["claude", "-p", user_prompt],
            env=env,
            capture_output=True,
            text=True
        )

        # --- PHASE 3: DEHYDRATION (Check-in) ---
        if result.returncode == 0:
            # If successful, save the new brain back to disk
            # We remove the old vault and replace it with the new state
            if os.path.exists(vault_path):
                shutil.rmtree(vault_path)
            shutil.copytree(execution_path, vault_path)
            
            return result.stdout
        else:
            # If it crashed, DO NOT overwrite the vault. 
            # Throw error but keep old memory safe.
            raise Exception(f"CLI Error: {result.stderr}")

    finally:
        # --- PHASE 4: CLEANUP ---
        # Always wipe the RAM execution folder
        if os.path.exists(execution_path):
            shutil.rmtree(execution_path)

import os
import shutil
import subprocess
from filelock import FileLock, Timeout

# Configuration
VAULT_ROOT = "/storage/sessions"
RAM_ROOT = "/dev/shm"

def run_synchronized_agent(session_id, user_prompt):
    # 1. Define Paths based on ID (Deterministic)
    vault_path = os.path.join(VAULT_ROOT, session_id)
    lock_path = os.path.join(VAULT_ROOT, f"{session_id}.lock")
    ram_path = os.path.join(RAM_ROOT, session_id)  # No random UUIDs!

    # 2. Acquire Lock (Waits up to 60s for other agents to finish)
    lock = FileLock(lock_path, timeout=60)

    try:
        with lock:
            # --- CRITICAL SECTION: We are the only process touching this ID ---
            
            # 3. Orphan Cleanup (Crash Recovery)
            if os.path.exists(ram_path):
                print(f"[Warn] Found orphaned RAM session {session_id}. Cleaning...")
                shutil.rmtree(ram_path)

            # 4. Hydrate (HDD -> RAM)
            if os.path.exists(vault_path):
                shutil.copytree(vault_path, ram_path)
            else:
                # Initialize new session from template
                shutil.copytree("/storage/templates/base_auth", ram_path)

            # 5. Run Execution (Fast!)
            env = os.environ.copy()
            env["CLAUDE_CONFIG_DIR"] = ram_path
            
            result = subprocess.run(
                ["claude", "-p", user_prompt],
                env=env,
                capture_output=True,
                text=True
            )

            # 6. Dehydrate (RAM -> HDD)
            # Only save if successful. If Claude crashed, we discard the RAM state.
            if result.returncode == 0:
                # Atomic swap or simple overwrite
                if os.path.exists(vault_path):
                    shutil.rmtree(vault_path)
                shutil.copytree(ram_path, vault_path)
                return result.stdout
            else:
                raise Exception(f"Claude Error: {result.stderr}")

    finally:
        # 7. Final Cleanup (Always runs, even on error)
        if os.path.exists(ram_path):
            shutil.rmtree(ram_path)
        # Lock releases automatically when 'with' block ends

    def run_tool(self, tool_type: ToolType, session_id: str, project_path: str, prompt: str):
        """
        Executes a CLI tool in an isolated, synchronized environment.
        
        :param tool_type: ToolType.GEMINI or ToolType.CLAUDE
        :param session_id: Unique ID for the conversation (e.g., "ticket_884")
        :param project_path: The physical path where files live (CWD)
        :param prompt: The instruction to send to the agent
        """
        
        # 1. Define Deterministic Paths
        vault_path = os.path.join(self.vault_root, f"{tool_type.value}_{session_id}")
        lock_path = os.path.join(self.vault_root, f"{tool_type.value}_{session_id}.lock")
        
        # Unique RAM path per RUN to absolutely prevent collisions if lock fails
        # (Though lock shouldn't fail, this is double-safety)
        run_uuid = str(uuid.uuid4())[:8]
        ram_path = os.path.join(self.ram_root, f"{tool_type.value}_{session_id}_{run_uuid}")

        # 2. Acquire Lock (Wait up to 60s)
        lock = FileLock(lock_path, timeout=60)
        
        try:
            with lock:
                logger.info(f"[{session_id}] Lock acquired. Preparing workbench...")
                
                # --- PHASE A: HYDRATION (Vault -> RAM) ---
                self._hydrate(tool_type, vault_path, ram_path)

                # --- PHASE B: EXECUTION (The Run) ---
                logger.info(f"[{session_id}] Executing {tool_type.value}...")
                output = self._execute_subprocess(tool_type, ram_path, project_path, prompt)

                # --- PHASE C: DEHYDRATION (RAM -> Vault) ---
                # Only save if execution was successful (output is valid)
                logger.info(f"[{session_id}] Success. Saving state to Vault...")
                self._dehydrate(vault_path, ram_path)
                
                return output

        except Timeout:
            logger.error(f"[{session_id}] Could not acquire lock for 60s. System busy.")
            raise RuntimeError("Agent System Busy - Lock Timeout")
            
        except subprocess.CalledProcessError as e:
            logger.error(f"[{session_id}] CLI Crashed: {e.stderr}")
            # CRITICAL: We do NOT dehydrate here. We discard the RAM state 
            # so the session reverts to the last known good state in the Vault.
            raise e
            
        finally:
            # --- PHASE D: CLEANUP ---
            if os.path.exists(ram_path):
                shutil.rmtree(ram_path)
                logger.debug(f"[{session_id}] RAM workbench cleaned.")

    def _hydrate(self, tool_type, vault_path, ram_path):
        """Copies session from HDD Vault to RAM. Creates new if missing."""
        if os.path.exists(vault_path):
            shutil.copytree(vault_path, ram_path)
        else:
            # Load Template (Contains Auth Keys)
            template_path = os.path.join(self.template_root, tool_type.value)
            if not os.path.exists(template_path):
                raise FileNotFoundError(f"Auth Template not found at {template_path}")
            shutil.copytree(template_path, ram_path)

    def _dehydrate(self, vault_path, ram_path):
        """Copies session from RAM back to HDD Vault (Atomic-ish replacement)."""
        # Remove old vault state
        if os.path.exists(vault_path):
            shutil.rmtree(vault_path)
        # Save new state
        shutil.copytree(ram_path, vault_path)

    def _execute_subprocess(self, tool_type, ram_path, project_path, prompt):
        """Runs the specific CLI command with environment isolation."""
        env = os.environ.copy()
        cmd = []

        if tool_type == ToolType.GEMINI:
            # Gemini Isolation: Spoof HOME directory
            env["HOME"] = ram_path
            # We assume 'gemini' executable is in the global PATH
            cmd = ["gemini", "-p", prompt]

        elif tool_type == ToolType.CLAUDE:
            # Claude Isolation: Point Config Dir to RAM
            env["CLAUDE_CONFIG_DIR"] = ram_path
            cmd = ["claude", "-p", prompt]

        # Execute
        result = subprocess.run(
            cmd,
            cwd=project_path,    # <--- The "Hands" (Filesystem Context)
            env=env,             # <--- The "Brain" (Memory Context)
            capture_output=True,
            text=True,
            check=True           # Raises CalledProcessError on non-zero exit
        )
        
        return result.stdout

How to Deploy This
Create Your Templates (One Time Setup) You need to create the "Base Images" for your agents that contain your API keys.

Bash
# 1. Create Template Folder
mkdir -p /storage/templates/claude
mkdir -p /storage/templates/gemini

# 2. Login once to generate the auth files, then copy them here
# (For Claude)
cp ~/.config/claude/claude.json /storage/templates/claude/
# (For Gemini)
cp -r ~/.gemini/* /storage/templates/gemini/
Run the Manager

The AgentCLIManager will now auto-clone these templates for every new session_id.

It will automatically handle the copy-to-RAM and copy-back-to-HDD.

It will prevent collisions using the file lock.

This gives you a robust, crash-resistant, multi-agent foundation using the CLIs.
